	Important note before Phase 4
	-----------------------------
	Phase 4 usually feels slower than Phase 3.

	Reason:
	- Phase 3 is a simple API sanity test.
	- Phase 4 uses legacy SAI-PTF style execution, which does more setup.
	- If pytest must be downgraded inside the container, package installation adds extra wait time.

	Best practice:
	- Run Phase 4 as separate commands, not one long chained command.
	- That makes progress visible and avoids the feeling that the terminal is stuck.
